# Finpro Typography

Typography plays a crucial role in creating a deliberate visual structure, helping users navigate through the hierarchy of information presented. In the Finpro Design System, the default font is Poppins, and you can utilize various typography variables in your code. However, it's essential to note that not all fonts have equivalent typography variants, and the Finpro system offers nine distinct typography options.

```css
.my-header {
  font: var(--fp-font-display-light);
}
```

## CSS Variables

The following CSS variables are available and utilized in all of the Finpro components to adhere to typography guidelines:

```css
--fp-font-family: 'PoppinsVariable', sans-serif;

/* Font weights */
--fp-font-weight-thin         : 100;
--fp-font-weight-extralight   : 200;
--fp-font-weight-light        : 300;
--fp-font-weight-regular      : 400;
--fp-font-weight-medium       : 500;
--fp-font-weight-semibold     : 600;
--fp-font-weight-bold         : 700;
--fp-font-weight-extrabold    : 800;
--fp-font-weight-black        : 900;

/* Font sizes */
--fp-font-size-6xs  : 0.063rem;   /* 1px */
--fp-font-size-5xs  : 0.125rem;   /* 2px */
--fp-font-size-4xs  : 0.25rem;    /* 4px */
--fp-font-size-3xs  : 0.375rem;   /* 6px */
--fp-font-size-2xs  : 0.5rem;     /* 8px */
--fp-font-size-xs   : 0.625rem;   /* 10px */
--fp-font-size-s    : 0.75rem;    /* 12px */
--fp-font-size-m    : 0.875rem;   /* 14px */
--fp-font-size-l    : 1rem;       /* 16px */
--fp-font-size-xl   : 1.25rem;    /* 20px */
--fp-font-size-2xl  : 1.5rem;     /* 24px */
--fp-font-size-3xl  : 1.75rem;    /* 28px */
--fp-font-size-4xl  : 2rem;       /* 32px */
--fp-font-size-5xl  : 3rem;       /* 48px */
--fp-font-size-6xl  : 4rem;       /* 64px */
```
