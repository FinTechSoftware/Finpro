# Breadcrumbs

Breadcrumbs treat any node as a child and place a separator between them.

<fp-alert variant="warning" icon>Sub-headings that should be spread as nodes should be given to the `fp-breadcrumbs` element in the form of an array as a property in the following format.</fp-alert>

<fp-alert variant="info" icon> <fp-breadcrumbs list=[{title: 'React', href: 'https://reactjs.org/'},{title: "Vue", href: 'https://vuejs.org/'},{title: 'Svelte', href: 'https://svelte.dev/'}] /></fp-alert>

## Font Size Breadcrumbs 

We have 18 sizes of breadcrumbs: **5xs**, **4xs**,  **3xs**, **2xs**, **xs**,  **s**,  **m**,  **l**,  **xl**,  **2xl**,  **3xl**, **4xl**, **5xl**, **6xl**,  **xxl**, **2xxl**, **3xxl**, **4xxl**. The default size is **xs**.

## Custom Style Breadcrumbs 

You can replace any or more of the children of the node you want with custom css.

<fp-alert variant="warning" icon>It can be manipulated by giving the `styling` property to the child of the node to be manipulated.</fp-alert>

<fp-alert variant="info" icon> <fp-breadcrumbs  list:[{title: 'React', href: 'https://reactjs.org/'},{title: "Vue", href: 'https://vuejs.org/', styling: 'color: red; background-color: blue'},{title: 'Svelte', href: 'https://svelte.dev/'}] /></fp-alert>
