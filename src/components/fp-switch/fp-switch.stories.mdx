# Switch

The switch component can be utilized to switch the On or Off state of a single item.

### Usage

To switch the state rapidly, you can use switch as a button for action.

* The action is triggered right away when you click on switch; users don’t need to click another button to save or apply the settings.
* Setting `aria-label` attribute is required when you use your own label outside this component.
* There is no indeterminate state for switch.
* Don’t make switch an input field; use checkbox for boolean inputs instead.
