# Dropdown Group
The Dropdown Group component is designed to be utilized within either the fp-dropdown or fp-dropdown-group component, and serves to exhibit an operation within these components.

export const Template = (args) => html`
    <fp-dropdown>
    <fp-dropdown-item>Action outside of Group</fp-dropdown-item>
  <fp-dropdown-group
    caption='${ifDefined(args.caption)}'
  ><fp-dropdown-item>Action 1</fp-dropdown-item><fp-dropdown-item>Action 1</fp-dropdown-item></fp-dropdown-group>
  </fp-dropdown>
`
