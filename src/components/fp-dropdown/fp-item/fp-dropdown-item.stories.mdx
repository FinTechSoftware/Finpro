# Dropdown Item
The Dropdown Item component is designed to be utilized within the fp-dropdown or fp-dropdown-group components, and its purpose is to present an actionable item within these parent components.

export const Template = (args) => html`
  <fp-dropdown-item
    icon='${ifDefined(args.icon)}'
  >Dropdown Item</fp-dropdown-item>
`
