# Select Option

The Select Option component is designed to be used within a Select component, and it is used to display a single option within the Select component.

export const Template = (args) => html`
  <fp-select-option
    value="fp"
    ?disabled='${args.disabled}'
    ?selected='${args.selected}'
  >finpro</fp-select-option>`;
