# Card

Cards can be used to group related subjects in a container.

## Basic Card

Basic cards aren't very exciting, but they can display any content you want them to.

<Canvas isRow>
    <Story name="Basic Card">
        {BasicCard.bind({})}
    </Story>
</Canvas>

## Card With Header

Headers can be used to display titles and more.

<Canvas isRow>
    <Story name="Card With Header">
        {CardWithHeader.bind({})}
    </Story>
</Canvas>

## Card With Footer

Footers can be used to display actions, summaries, or other relevant content.

<Canvas isRow>
    <Story name="Card With Footer">
        {CardWithFooter.bind({})}
    </Story>
</Canvas>


## Images

Cards accept an image slot. The image is displayed atop the card and stretches to fit.

<Canvas isRow>
    <Story name="Card Images">
        {CardImage.bind({})}
    </Story>
</Canvas>


## Cards Variants

We have 8 variant for each card: **Primary**, **Secondary**, **Tertiary**, **Alternative**, **Featured**, **Warning**, **Success** and **Danger**.

<Canvas isRow>
    <Story name="Variants">
        {CardTypeTemplate.bind({})}
    </Story>
</Canvas>

## Card Border Radius

We have 8 border radius type for each card: **n**, **2xs**, **xs**, **s**, **m**, **l**, **xl**, **2xl** and **pill**.

<Canvas isRow>
    <Story name="Border Radius">
        {CardBorderRadius.bind({})}
    </Story>
</Canvas>
