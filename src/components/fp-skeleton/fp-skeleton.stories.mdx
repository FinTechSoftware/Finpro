# Skeleton

A skeleton indicator is a component used as a preview that components are loaded for a long process or a process that can take significant or unknown time.

<fp-alert variant="warning" icon>The inline styles in the examples are for **demo purposes** only. Use regular CSS classes or tag selectors to set styles.</fp-alert>

## Basic Skeleton

By default, `width` is 100, `height` is 10. So the width and height must be a number between 0-100. `radius` values can take a value between xs and 4xxl, by default they take the value `xs`. The component to be shaded is obtained by entering the desired height and width. The `animation` tag is used to activate the animation of the skeleton.

## Skeleton Sizes

We have 14 different sizes of skeleton radius: `xs`, `s`, `m`, `l`, `xl`, `2xl`, `3xl`, `4xl`, `5xl`, `6xl`, `xxl`, `2xxl`, `3xxl`, `4xxl`. The default size is `xs`.

## Primary Variants

Primary skeleton are used for main actions of the screens. It can has 4 different "kind"s. `primary`, `secondary`, `success` and `danger`.
