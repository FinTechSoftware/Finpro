## Checkbox Menu Items
By setting the type attribute to checkbox, you can create a menu item that will toggle on and off when clicked. You can also use the checked attribute to specify the initial state of the checkbox.

Checkbox menu items look the same as regular menu items and don't have any visual distinction. Their toggle capability is usually understood from the context, similar to how it works in the menus of native apps.

## Value & Selection

If you want to assign a hidden value, like a unique identifier, to a menu item, you can use the value attribute. Once an item is selected, the sl-select event is emitted, and you can access the selected item's reference at event.detail.item. This reference allows you to retrieve the selected item's value, checked state, and other properties.
